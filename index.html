Background
How this started

These tools were built to solve real problems encountered on the job. Instead of working around missing or overcomplicated software, I started building small focused utilities that do exactly what's needed — nothing more.

AI tools made it possible to build and iterate faster than ever. Each tool in this suite was developed with AI assistance, then tested and refined in actual production environments. The results felt worth sharing, so I published them here — free to use for anyone in the field.
