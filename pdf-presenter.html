PDF Presenter

v1.0 · Web app

A browser-based presenter tool for PDF slideshows. Dual-window mode with real-time sync, per-slide notes, resizable layout, and black screen control — delivered as a single HTML file, no installation required.

Download .html Manual
Overview
How it works
Dual-window mode
The presenter window runs on your laptop; the output window goes to the external display, media server, or video mixer input. Both stay in sync with zero perceptible lag.
Presenter layout
Current slide on the left, next-slide preview and notes on the right. Drag the vertical divider to balance the two areas; drag the horizontal divider to balance preview and notes.
Per-slide notes
Type notes for each slide directly in the interface. In folder mode, notes are saved automatically to a plain-text file alongside the PDF — no manual action required.
Folder mode
Open an entire event folder to get automatic note saving and layout persistence per PDF file. Switch between PDFs from the file list without losing any data.
Black screen
Press B or click the button to instantly replace the output with a solid black frame. The presenter view stays active so you can prepare the next action discreetly.
No installation
A single HTML file — open it in Chrome or Edge and it's ready to use. No server, no internet connection required after the first load. PDF.js is cached by the browser on first use.
