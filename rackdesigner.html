Rack Designer

v0.1.18 · Windows 10/11

A visual rack layout application for live events, touring productions, permanent installations, and broadcast environments. Build rack layouts, manage a device library, and export rider-ready PDF documentation.

Download setup Manual
Overview
How it works
Visual rack layout
Design multi-rack layouts by dragging equipment from a library into rack slots. Supports reordering by drag, tray placement for sub-width devices, and multi-bay racks.
Device library
Manage a persistent, project-independent library of equipment definitions. Includes 100+ preloaded professional devices across video, audio, lighting, network, and power categories.
Power & weight totals
Automatically calculates total power draw in watts and weight in kg per rack and across the entire project. Totals update in real time as devices are added or removed.
PDF export
Generate rider-ready PDF documentation with rack diagrams, device lists, power and weight totals, and a project summary. Fully configurable per export.
Import & export
Import and export devices in JSON or CSV format. Share library entries with other users or maintain a master library outside the application.
Project files
Projects are saved as self-contained .rackproj files. Share them with collaborators without requiring the same device library to be installed on their machine.

Contribute to the library

The preloaded library covers the most common equipment, but the community can expand it. Share your device entries with other users via JSON export — open an issue or start a discussion on GitHub.
